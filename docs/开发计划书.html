游戏流程

从初始化到结果展示的完整流程

1. 初始化

加载场景，初始化 GameState

2. 等待放置

显示地图，玩家点击放置猫塔

3. 波次开始

点击"开始"按钮，启动波次

4. 敌人生成

每秒生成一个敌人，共 10 个

5. 敌人移动

沿路径点向终点移动

6. 塔攻击

每秒对范围内敌人发射子弹

7. 子弹命中

子弹到达敌人位置，造成伤害

8. 死亡处理

敌人血量归零，移除并播放动画

9. 结束判断

胜利：10 个敌人生成且全部死亡
失败：任意敌人到达终点

10. 结果显示

显示 Victory! 或 Defeat!

验收标准

功能完成的判断条件

  • ✓ 玩家可以点击地图放置猫塔（仅限一次）
  • ✓ 点击"开始"按钮后，敌人从左侧进入并沿路径移动
  • ✓ 猫塔自动攻击范围内敌人
  • ✓ 子弹命中敌人造成伤害，敌人血条减少
  • ✓ 敌人死亡后移除
  • ✓ 敌人到达终点显示 "Defeat!"
  • ✓ 所有敌人生成且死亡后显示 "Victory!"
  • ✓ 微信小游戏可运行（主包 < 4MB）
